# Command: /code-search

## Purpose
Search for information using semantic code search. Find information, code patterns, or documentation using semantic search.

## Usage
```
/code-search [search terms]
```

## Arguments
- `search terms`: What to search for

## Examples
```
/code-search "authentication patterns"
/code-search "error handling in API"
```

## Workflow Integration
- **Before**: Formulate specific search terms
- **During**: Use Serena semantic search
- **After**: Use results for implementation or debugging

## Implementation Notes
Search for information about $ARGUMENTS using Serena's semantic search capabilities.
